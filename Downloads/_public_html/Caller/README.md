# AdmissionConnect - MySQL Version

Complete telecaller management system for Hostinger.

## Quick Setup

1. Create MySQL database in Hostinger hPanel
2. Import `database.sql` via phpMyAdmin
3. Upload all files to `public_html/`
4. Edit `includes/config.php` - Update DB credentials & BASE_URL
5. Visit `setup_admin.php` - Creates admin user
6. DELETE `setup_admin.php`
7. Login at index.php

✅ Auto-assign students to telecallers
